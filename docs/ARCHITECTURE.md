# MoonEventSim 确定性事件驱动仿真框架架构白皮书

## 1. 核心设计理念
`MoonEventSim` 是专为分布式系统、网络协议与并发算法设计的确定性离散事件仿真（DES）框架。通过虚拟时间引擎消除操作系统调度非确定性，提供 100% 可复现的虚拟实验室。

## 2. 模块分层架构
- **核心时间引擎 (`core`)**：基于纳秒级虚拟时间轴与最小堆优先队列，提供零抖动的时间推进机制。
- **协作式进程模型 (`process`)**：提供轻量级虚拟 Actor 调度原语。
- **确定性同步管道 (`sync`)**：实现虚拟时钟下的 `Channel`、`Mutex`、`Semaphore` 与 `Barrier`。
- **网络拓扑与故障注入 (`net` & `fault`)**：支持丢包、乱序、网络分区与重试熔断仿真。
- **时序统计监控 (`metrics`)**：实时统计吞吐量，支持 P50, P90, P95, P99 尾部延时计算。
